Subset of Preliminary Grid Study Results¶

For the case of no scattering, we can compute the true analytical solution against which all other results can be compared. However, when scattering is introduced, we no longer have that luxury.

Therefore, in order to verify that a FD solution is reliable, the best that we can do is compare it to other FD solutions of different resolutions. In the plots, below, I compare the convergence behavior of two grid studies in the absence of scattering in order to know how to identify a converged solution when scattering is introduced and the true solution is no longer known.

In the first case, the maximum resolution resolution run is sufficiently large (88 spatial x 6 angular) to consider it converged (within ~1% of the true solution). In the second case, the maximum resolution is insufficient (23 spatial x 10 angular) to be considered converged (within ~4% of true solution).

In both cases, we have:

a_water = 0.1
b = 0.0
kelp_dist: top-heavy

Observations¶

  • Both FD vs. FD plots show that the second-best solution has an average relative error of significantly less than 1e-2. These values do not seem directly related to the actual error from the true solution.
  • In the "non-convergent" case, the error vs. ns plot becomes non-linear for large resolutions. The nonlinearity seems to begin when errors are near 4%, which is the actual error. I'm not sure if this is a coincidence, or if it may be a reliable way of determining proximity to the true solution.

Caveats¶

  • I realize that 4% is not very convincing as a "non-convergent solution". I'll retry with a smaller, less accurate grid.
  • After some thought, I don't think I'm actually plotting "average relative error". It's more like a rescaled absolute error in the sense that I'm dividing all absolute error values by the same norm. In a proper relative error, the division should happen pointwise.
